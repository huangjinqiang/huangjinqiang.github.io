[只出现一次的数字](https://leetcode-cn.com/leetbook/read/top-interview-questions/xm0u83/)


### 题意
* 非空整数数组，只有一个元素单独出现，其他元素成对出现
* 具有线性时间复杂度，不使用额外空间

### 异或
* 相同为0，不同为1，由相同为0可知 ```a ^ a = 0```
* 满足交换律 ```a ^ b ^ c = a ^ (b ^ c)```

### 代码
```js
/**
 * @param {number[]} nums
 * @return {number}
 */
var singleNumber = function(nums) {
  return nums.reduce((a, b) => a ^ b)
}
```
